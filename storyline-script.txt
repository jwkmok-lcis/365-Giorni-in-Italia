365 Giorni in Italia — Storyline Script

=== GRAMMAR SKILL SYSTEM ===

🎯 Core Concept
XP is earned not just for talking to NPCs, but for demonstrating mastery of specific grammar structures embedded in dialogue choices. Each choice has a skill tag [S#] that tracks which grammar competency was used.

⸻

📊 SKILL TIERS (Progressive Difficulty)

🟢 A1 Skills (Chapter 1–3, Days 1–7)
 S1: Basic survival phrases
 S2: Present tense conjugation (io cerco, tu vai, lui fa)
 S3: Articles & gender (il mercato, la ricetta, un pane, una carota)
 S4: Simple questions (Dove? Quando? Quanto? Perché?)
 S5: Vocabulary repetition & recognition (ragù, soffritto, fuoco basso)

🟡 A2 Skills (Chapter 2+, Days 4+)
 S6: Passato prossimo (ho visto, ha detto, ho trovato)
 S7: Connectors (ma, perché, quindi, se)
 S8: Object pronouns (lo, la, mi, ti)
 S9: Modal verbs (posso, devo, voglio, posso)
 S10: Infinitives after modals (voglio vederlo, devo parlare)

⸻

⚙️ XP AWARD MECHANICS

1. Choice Accuracy Tiers
   ✅ Correct form → Full XP
   ⚠️ Acceptable (minor error) → 50% XP
   ❌ Incorrect → 0 XP + NPC rephrasing offered

2. XP Values (Examples)
   Recognizing S5 structure → +3 XP
   Producing S5 structure → +5 XP
   Producing S6 (past tense) → +10 XP
   Combining two skills (e.g., S7+S9) → +15 XP
   Complex combinations (S6+S7+S9) → +20 XP

3. Repetition Bonus
   Use same pattern 3x correctly in same day → +10 bonus XP
   Pattern: "Ho parlato", "Ho trovato", "Ho capito" = bonus triggered
   OR: Use 2x different patterns from same skill → +8 bonus XP
   Pattern: "Perché?" + "Ma..." = S7 bonus

4. Understand vs Produce Split
   Comprehension (reading/listening NPC) → +2 XP base
   Production (choosing/building response) → +8 XP base

5. Daily Skill Focus
   Day 1: S5 (vocabulary, basic present)
   Day 2: S5 (repetition, recognition)
   Day 3: S5 + S4 (question formation)
   Day 4: S6 (passato prossimo intro)
   Day 5: S7 (connectors: ma, perché, quindi)
   Day 6: S9 (modals: posso, devo, voglio)
   Day 7: S7+S9 (combining connectors + modals)
   Day 8: S8 (object pronouns: lo, la, mi, ti)
   Day 9: S6+S8 (past + pronouns)
   Day 10: S7+S8 (connectors + pronouns)
   Day 11: S8+S9 (pronouns + modals)
   Day 12: S6+S7+S8 (complex past + connectors + pronouns)
   Day 13: S9+S10 (modals + infinitives)
   Day 14: S7+S9+S10 (full synthesis)
   Day 15: S8+S9 (pronouns + modals reinforcement)
   Day 16: S6+S7+S8 (complex combinations)
   Day 17: S9+S10 (modals + infinitives)
   Day 18: S7+S9+S10 (full synthesis)
   Day 19: S8 review (object pronouns in new contexts)
   Day 20: S6+S8 (past + pronouns)
   Day 21: S7+S8 (connectors + pronouns)
   Day 22: S8+S9 (pronouns + modals)
   Day 23: S6+S7+S8 (complex)
   Day 24: S9+S10 (modals + infinitives)
   Day 25: S7+S9+S10 (synthesis)
   Day 26: S5-S8 review (vocabulary + pronouns)
   Day 27: S6+S9 (past + modals)
   Day 28: S7+S8+S9 (triple combination)
   Day 29: S6+S7+S8+S9 (full integration)
   Day 30: S10 mastery (infinitives in complex contexts)

6. Error Feedback Loop (Gamified Learning)
   Player chooses → If ❌ incorrect
   NPC: "Non capisco. Intendi dire: [correct form]?"
   Player retries or continues → +XP on correct attempt
   (No punishment, just reframing)

   Example Error Branch:
   NPC: "Hai parlato con Giorgio?"
   Player chooses: "Sì, Giorgio." [❌ Incomplete]
   NPC: "Non capisco. Intendi dire: 'Sì, ho parlato con Giorgio'?"
   Player retries: "Sì, ho parlato con Giorgio." [✅ +10 XP]

7. Skill Mastery Milestone
   Skill reaches Level 3 → +20 XP milestone bonus
   Triggers when: 5 correct uses of skill in varied contexts

⸻

🎮 EXPECTED PROGRESSION (Example)
   Day 1, Marco node:
   Player sees: "Buongiorno! Cerchi qualcosa?"
   Choice: "Sì, cerco il ragù." [S5 production]
   Result: +5 XP (S5 Level 1/3)
   
   By Day 3, after 15+ S5 uses:
   S5 at Level 3 → +20 milestone bonus
   Skill unlocked for next tier

   Day 4, Lucia node:
   NPC: "Hai parlato con Giorgio?"
   Choice: "Sì, ho parlato con lui." [S6 production]
   Result: +10 XP (S6 Level 1/3)
   Pattern builds naturally from context
